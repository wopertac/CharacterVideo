<h2>This Python script remakes your video from symbols.</h2>

To work with it you need to install PIL, cv2, yt_dlp and numpy
```bash
pip install pillow opencv-python yt-dlp numpy
```

How to use it:


For use with your camera:
```bash
python main.py
```
For use with mp4 video:
```bash
python main.py -v your_video.mp4
```
For use with youtube link:
```bash
python main.py -l link_on_youtube_video
```
